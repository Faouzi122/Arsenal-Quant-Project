# Compétences et Outils Techniques

## Langages et Frameworks
* **Python Expert :** Maîtrise avancée du langage pour les bots et les APIs.
* **FastAPI / Flask :** Création d'APIs légères et performantes pour RapidAPI.
* **SQLAlchemy / TinyDB :** Gestion de bases de données (locales et légères).

## Architecture
* **Clean Architecture :** Séparation des couches (Entités, Cas d'usage, Adaptateurs).
* **Design Patterns :** Utilisation pertinente des patrons (Factory, Singleton, Observer).

## Automatisation
* **Interfaçage API :** Capacité à connecter des services tiers (Binance, RapidAPI, etc.).* **System Automation :** Capacité à orchestrer le système de fichiers via scripts Bash et Python subprocess.
